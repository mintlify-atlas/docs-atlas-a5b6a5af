## What is FTB Hustle?

FTB Hustle is a platform designed specifically for ambitious students who want to level up their careers. We help you discover hackathons, grants, competitions, internships, scholarships, and curated learning resources - all in one place.

<Note>
  **Built for students, by students.** Every feature is designed to help you find opportunities faster, stay organized, and never miss a deadline.
</Note>

## Key Features

### Personalized Opportunity Feed

After a quick onboarding (~60 seconds), we personalize your feed based on:
- Your role (student or society member)
- Education level and field of study
- Location preferences (state or city)
- Opportunity interests (internships, hackathons, scholarships, etc.)
- Domain preferences (AI/ML, Web Dev, Product Management, etc.)

```typescript
// Your profile shapes what you see
const profile = {
  persona: "student",
  educationLevel: "Undergraduate",
  fieldOfStudy: "Computer Science",
  opportunityInterests: ["Internships", "Hackathons", "Scholarships"],
  domainPreferences: ["AI/ML", "Web Development"],
  location: "Maharashtra"
}
```

### Smart Application Tracker

Keep track of every opportunity you're pursuing:
- Bookmark opportunities with one click
- Track application status (Not Applied, Applied, Interview, Offer, Rejected)
- Set custom notes and reminders
- View all deadlines in one place
- Sync with calendar events

### Curated Learning Toolkits

Access structured learning paths created by industry experts:
- Video lessons and tutorials
- Downloadable resources and templates
- Progress tracking across lessons
- Premium content with verification codes
- Organized by topic and difficulty level

### Ungatekeep Resources

Get insider knowledge and resources that typically stay hidden:
- Resume templates and examples
- Interview preparation guides
- Portfolio projects and case studies
- Career roadmaps and skill trees
- Community-shared tips and tricks

## Platform Tech Stack

<Note>
  FTB Hustle is built with modern, performant technologies for the best user experience.
</Note>

Our platform is powered by:
- **Next.js 15** with App Router for blazing-fast navigation
- **TypeScript** for type-safe development
- **Better Auth** for secure authentication with Google and LinkedIn OAuth
- **PostgreSQL + Drizzle ORM** for reliable data management
- **Tailwind CSS 4** for beautiful, responsive design
- **React Query** for efficient data fetching and caching
- **Radix UI** for accessible component primitives
